Responsive

Responsive behaviour is built-in and doesn’t require any custom development, extremely easy. Layouting is automatically done based on screen size while making sure all widgets used all of the available space.

Simple and easy to use API

The vaadin-board is divided into rows using vaadin-board-row. Within a row you can have any Web Component or HTML elements inside it. Sizing, layouting and wrapping is automatic.

Up to four columns supported

Vaadin Board supports one to four columns on each row.

Nested rows

You can have rows inside rows for more fine grained control on how certain areas of your layout behave on resize and how they are rendered.

Column span

An element inside a vaadin-board-row can have a board-cols modifier for controlling the area used within the row. Choosing values of 2 or 3 you get to layout rows such as 25%-50%-25%, 33%-67% or 75%-25%.
